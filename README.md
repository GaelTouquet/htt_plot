# htt_plot

## Installation

* Go to lyovis10
* Clone this directory
* Put this package in your python path by doing: 

```
source ./init.sh
```

* Copy your small root files in this package, in a directory called `lucas_small` (see [components/lucas_small.py](components/lucas_small.py))
